# Cosmic Conquest
Cosmic Conquest is a 2D Platformer Game built by students, for the students. It is a combat game where user has to navigate the map and defeat enemies in order to survive. Collect as many coins to get the highest score.
## Installation
Download all the files from the Google Drive. Extract all the files and run Cosmic Conquest.exe.
## Features
* Player movements and respawns
* Projectile shooting and killings of enemies
* Enemies damage
* Ladders climbing
* Level advancements  
## Usage
* Key A and D for Left and Right movements
* Spacebar to jump
* Key W and S for up and down movements (when using the ladder)
* Left mouse button to fire
## Tech Stack
Unity, GitHub, C#, Aseprite
